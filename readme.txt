Questo plugin ci aiuta a bloccare i commenti che vengono segnalati come spam dagli utenti


== Description ==
Segnalando un commento come spam, il nostro plugin bloccherà il commento ed eviterà che possa ritornare nuovamente.
Questo plugin lavora e funziona solo sui servizi hosting di Iweblab.it

== Installation ==
Per installare il plugin:

1. Decomprimi il file 'block-spam-comment-iweblab.zip' e carica la cartella 'spam-ip-export' nella directory 'wp-content/plugins/' della tua installazione di WordPress.
2. Attiva il plugin dalla pagina "Plugins" del menù di WordPress.
